# LairaBudget 💸

LairaBudget is a personal finance dashboard designed to help users manage their money in a simple and visual way.

## 🚀 Features

- Monthly budget control
- Savings tracking (cofres)
- Expense tracking with categories
- Daily spending limit calculation
- Financial insights & charts
- Trip & leisure budgeting
- PWA support

## 🧠 Tech

- JavaScript
- Chart.js
- LocalStorage
- PWA

## 📌 Status

In development – currently improving UI/UX.
